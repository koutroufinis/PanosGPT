# PanosGPT 🚀

A beautiful, responsive chat interface powered by Groq AI with real-time typing animation and markdown support.

## ✨ Features

- 💬 **Real-time chat** with Groq AI (openai/gpt-oss-120b)
- ⌨️ **Typing animation** that mimics human typing
- 📝 **Markdown support** for code blocks, headers, lists, and more
- 🌐 **Language toggle** (English/Greek)
- 📱 **Fully responsive** - works on all devices
- 🎨 **Dark theme** with beautiful styling
- ⏹ **Stop button** to interrupt responses
- 🔒 **Secure API key management**

## 🚀 Quick Start
